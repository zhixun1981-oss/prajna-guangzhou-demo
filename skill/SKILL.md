# prajna-salary-template

## 技能定位

本技能是 Prajna 企智的 **薪资模板 Agent 原生能力**，面向各行各业、各层级岗位，一键生成结构完整、公式联动的 Excel 薪资模板。

无论是电商运营、软件工程师、生产主管、门店店长，还是金融客户经理、餐厅经理、行政专员，均可通过本技能快速生成符合岗位特性的薪资结构、绩效考核、调薪机制、奖金方案、福利明细、计算示例与合规校验。

## 触发条件

- “生成薪资模板 / 做一份薪资表”
- “帮我做一份 XX 岗位的薪酬方案”
- “薪资结构表 / 绩效考核表 / 奖金方案表 / 合规校验表”
- “HR 薪酬体系模板 / 岗位薪资测算 / 批量生成薪资模板”
- `prajna skill run prajna-salary-template --preset=互联网-电商运营助理`

## 执行流程

1. **识别岗位与行业**
   - 从用户输入提取：行业、岗位、职级、城市、薪资水平、自然语言描述等关键信息。
   - 若用户未指定，使用通用模板或推荐最匹配的预设。

2. **加载多维参数**
   - 内置 10 套主流行业/岗位预设参数（互联网、制造业、零售、金融、餐饮、行政等）。
   - 城市薪酬基准：自动根据城市等级调整薪资、社保公积金基数。
   - 职级带宽：根据 P1-P9 职级套用对应的薪酬带宽与奖金带宽。
   - 自然语言岗位推断：通过规则引擎识别用户描述中的行业与岗位类型，套用模板。
   - 现有薪酬表补全：解析用户上传的薪酬表，自动识别字段并填充到模板中。

3. **生成完整工作簿（6+ 工作表）**
   - **薪资结构**：基本工资、岗位工资、绩效工资、加班工资、补贴、五险一金、个税、应发/实发工资，以及薪酬带宽。
   - **绩效考核**：6-8 个岗位 KPI 维度、权重、目标值、评分标准、数据来源。
   - **调薪机制**：试用期转正、年度普调、晋升调薪、绩效调薪、市场调整、特殊贡献、留任调薪。
   - **奖金方案**：月度提成、月度绩效奖金、季度奖金、年度奖金、项目奖金、全勤奖、伯乐奖、年终分红。
   - **福利明细**：五险一金、补充医疗、带薪年假、节日/生日福利、培训补贴、团建、体检、各类津贴。
   - **计算示例**：完整员工薪资计算案例，含绩效得分、五险一金、个税、实发工资、全年总包预估、图表可视化。
   - **合规校验**：最低工资、社保基数上下限、公积金比例、绩效工资占比、个税、实发工资等自动校验。
   - **绩效系数对照表**：绩效得分区间与绩效系数对应关系。

4. **公式化输出**
   - 所有汇总、绩效加权、个税、全年总包、合规结果均使用 Excel 公式，支持修改输入后自动重算。

5. **可选增强输出**
   - 生成 Word/PDF 版本的薪资说明文档（依赖 `python-docx` / `reportlab`，缺失时自动跳过）。
   - 批量生成多个岗位模板，并输出汇总表。
   - 保存历史版本，支持同岗位不同版本参数对比。
   - 使用 `recalc.py` 校验所有公式语法、引用与计算结果。

## CLI 参数

| 参数 | 说明 |
|---|---|
| `--preset` | 使用内置预设名称 |
| `--industry` | 行业（如“互联网”） |
| `--position` | 岗位（如“产品经理”） |
| `--city` | 城市，应用城市薪酬系数 |
| `--level` | 职级 P1-P9，应用职级带宽 |
| `--description` | 自然语言岗位描述 |
| `--input` | 现有薪酬表 `.xlsx` 路径，自动补全字段 |
| `--output` | 输出路径或目录（默认 `~/.prajna/salary_template_history/`） |
| `--format` | `excel` / `word` / `pdf` / `all` |
| `--batch` | 批量 CSV 路径（列：`industry,position,city,level,output_name`） |
| `--history` | 保存到历史版本库 |
| `--compare` | 对比最近两份同岗位历史记录 |
| `--compare-position` | 指定对比的岗位名称 |
| `--list-presets` | 列出所有预设 |

## 使用示例

```bash
# 列出所有预设
python /Users/a12345/prajna/skills/hr/prajna-salary-template/scripts/generate_salary_template.py --list-presets

# 使用预设生成（广州、P2 职级）
prajna skill run prajna-salary-template --preset=互联网-电商运营助理 --city=广州 --level=P2 --history

# 自定义行业和岗位
prajna chat -q "帮我做一份新能源汽车行业电池工程师的薪资结构表。"
prajna skill run prajna-salary-template --industry=新能源汽车 --position=电池工程师 --city=深圳 --level=P3

# 自然语言描述
prajna skill run prajna-salary-template --description "深圳新能源电池pack工艺工程师" --history

# 上传现有薪酬表自动补全
prajna skill run prajna-salary-template --input=现有薪酬表.xlsx --industry=互联网 --position=产品经理 --city=广州

# 批量生成
prajna skill run prajna-salary-template --batch=岗位清单.csv

# 同时输出 Word/PDF
prajna skill run prajna-salary-template --preset=互联网-电商运营助理 --format=all

# 对比历史版本
prajna skill run prajna-salary-template --compare --compare-position=电商运营助理

# 校验已生成文件公式
python /Users/a12345/prajna/skills/hr/prajna-salary-template/scripts/recalc.py prajna_薪资模板_*.xlsx
```

## 输出约定

- 默认输出目录：`~/.prajna/salary_template_history/`
- 文件命名：`prajna_薪资模板_<行业>_<岗位>_<日期>.xlsx`
- 历史版本命名：`~/.prajna/salary_template_history/<时间戳>_<原文件名>`
- 文件类型：Excel（8 个工作表），可选 Word / PDF
- 所有涉及法律/合规/敏感决策的输出，需附加免责声明并建议人工复核

## 协同调用规范（与现有 Skill 打通）

本 skill 作为薪酬体系生成中台，被以下 Skill 在关键节点显式调用。调用方式统一为：识别岗位/行业参数 → 调用 `scripts/generate_salary_template.py` → 输出 Excel 薪资模板。

| 调用方 Skill | 调用节点 | 输入 | 输出物 |
|---|---|---|---|
| `prajna-hr-methodologies` | 岗位价值评估完成后、薪酬带宽设计时 | 岗位名称、职等、行业 | 薪资结构表、薪酬带宽参考 |
| `prajna-hr-payroll` | 薪酬核算体系搭建、薪资测算时 | 岗位、城市、薪资水平 | 完整薪资测算模板 |
| `prajna-performance` | 绩效方案设计完成后、奖金方案配套时 | 岗位、KPI 维度 | 绩效与奖金联动模板 |
| `prajna-recruitment-package` | JD 生成后、Offer 薪酬沟通时 | 目标岗位 | 薪资结构参考表 |
| `prajna-report-agent` | 薪酬分析、人力成本报表时 | 岗位范围 | 薪资模板基础表 |
| `prajna-finance` | 人力成本预算、薪酬总包测算时 | 岗位、人数 | 薪酬预算测算模板 |

### 调用约定

1. **参数传递**：调用方负责收集岗位、行业、职级、薪资水平等变量，作为 `--preset` 或 `--industry`/`--position` 传入脚本。
2. **输出承接**：本 skill 输出 Excel 文件路径，调用方可直接交付用户或进一步嵌入报告。
3. **免责声明**：所有输出必须附加 `[人工智能生成-需人工核验]` 声明，薪酬决策建议人工复核。
4. **失败回退**：若本 skill 调用失败，调用方应使用自身默认流程继续，不影响主任务完成。
